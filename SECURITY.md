# Security Policy

If you believe you have found a security vulnerability in `black-tonny-frontend`, please do not disclose it through a public GitHub issue.

## Reporting

Send the report to:

- `lizhihui0215@gmail.com`

Please include:

- affected repository and branch
- impact summary
- reproduction steps or proof of concept
- relevant logs, screenshots, or payload samples if available

## Response

Reports will be reviewed and responded to as reasonably possible.
