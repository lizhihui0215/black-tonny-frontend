# @vben/backend-mock

Business mock service for `black-tonny-frontend`, aligned to the official `vue-vben-admin v5` `apps/backend-mock` pattern.

It provides the repo-owned mock routes and export fixtures that frontend development and local frontend/backend parallel work rely on before the sibling backend takes over the same formal `/api/*` paths.

`apps/backend-mock` itself is a normal integrated dev capability in the `vben` baseline. The ongoing correction target in this repo is not whether this package should exist, but whether its route shape matches the sibling backend API closely enough.
